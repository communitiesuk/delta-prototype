Help with status
  • Organisations the number of organisations who have been asked to submit a return for this collection.
  • Not started: the number of organisations who have not create a record.
  • In draft: the number of organisations who have started at least one record and saved it in draft format.
  • Awaiting certification: the number of organisations with a return that has records that need to be certified.
  • Completed: the number organisations that have submitted and certified a return. [OR for uncertifiable returns] the number of organisations who have submitted a return.
  • Extended: flags whether the submission deadline for this return has been extended beyond the original collection window.
