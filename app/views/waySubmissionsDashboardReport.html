Overview

Help with labels and removed organisations
  • Labels: :
    • Organisations: The number of organisations that have been asked to submit a return for this collection.
    • Extended: Flags whether the submission deadline for this return has been extended beyond the original collection window.
    • Not started: The number of organisations that have not created a record.
    • In draft: The number of organisations that have started at least one record and saved it in draft format.
    • Awaiting certification: The number of organisations with a return that has records needing certification.
    • Completed: The number of organisations that have submitted and certified a return. [OR for uncertifiable returns] The number of organisations that have submitted a return.
  • Include removed organisations: This option includes organisations that have submitted a return but were later removed from the collection group. These organisations will be included in the report counts, such as the organisation count.

Returns by organisation

Help with table labels
  • Organisation: The name of the organisation submitting the return.
  • Return status: The status of the return, based on the least progressed record within it. For example, if a return has 4 records, with 3 completed and 1 in draft, the return status will be 'draft'.
  • Deadline: The deadline for submitting the return. There will be an 'extended' tag next to the date if an extension has been granted.
  • Updated: The most recent date that any record within the return was edited.
  • Records: The number of records that match the selected filters. If no filters are selected, this shows the total number of records.
