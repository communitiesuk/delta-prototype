Jobs to be done (Collections)

External users

Data provider

  • Find the collection I need to provide a return for
  • Understand what period of time the collection relates to
  • Understand collection deadline
  • Submit a return (record(s))
  • View previous collection returns (check records)
  • View collections I may have missed
  • Understand who else can / has submitted data for a collection
  • Find any collections I have access to
  • View if there are other collections I should have access to / join
  • Change my role (provider and certifier)
  • Manage my account (including change my organisation)

Data certifier

  • View which records I need to certify
  • Certify a record
  • Contact the record owner (submitter) if data is incorrect and/or I have questions
