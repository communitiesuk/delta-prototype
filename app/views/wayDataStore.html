Help with table labels
  • Collection name: The title of the collection.
  • Status:
    • Future: The collection is currently closed and will open in the future.
    • Open: The collection is open, and there are no extensions.
    • Extended: The collection is open and there is an extension for one or more organisations.
    • Closed: The collection is closed, and returns cannot be submitted.
  • Date open: The date the collection opens, and returns can be submitted.
  • Deadline: The last date any organisation can submit a return. This includes extensions.
  • Organisation: The number of organisations that can submit a return.
  • Not started: The number of organisations that have not started a return.
  • In progress: The number of organisations that have records either in draft or that need to be certified.
  • Completed: The number of returns that have either been certified or does not need to be certified.
