UX design and spec for data range selector for us in filters throughout Delta

https://mhclgdigital.atlassian.net/browse/DT-1292

Data picker requirements

  • User can specify day*, month, year
  • Date follow format: Day Month Year.
  • Months are spelt out. Years are always 4 digits. Example: 1 January 2923 or 31 October 2024.
  • For Delta where possible we want to allow users to avoid having to type in data. Instead we want to offer drop-down controls.
  • Calendar date picker
  • Should be option to clear data filter
  • Needs to be accessible
  • Needs error reporting (if date input required)
*Optional

Data range selector requirements

  • Note
  • Calendar date picker
  • Error checking
    • From and to dates exist
    • To date is after from date
