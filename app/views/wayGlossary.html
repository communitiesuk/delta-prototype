Delta improvements

New names

We have updated some names to make them consistent across Delta and easier to understand.

The hierarchy is:

Collection Group > Collection > Return > Record

Collection group

This provides information on the collection, what it is for, how the data is used and lists all collections, both past and present.
Example: Hedgehog Census
Join collection groups and manage your roles

Collection

This is a one-off request for a return.
Example: Hedgehog Census 2023
View your collections

Return

This is the data submitted for a collection. A return may contain one or more records, depending on the collection type.
Example: Waverley Hedgehog Census 2023

Record

This is a set of data forming part of a return.
Example: Godalming
