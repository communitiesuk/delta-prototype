Delta Collections Dashboard (DCD)

  • Internal reporting tool - high level, at a glance
  • Page (dashboard) consists of two parts: annual chart (schedule), tabulated data

What and why

  • Primary
    • So we can see which collections are live or opening (coming) soon
  • Secondary
    • Understand (get a sense of) size, responses, usage
    • Link to where live collection (form) can be viewed) - collection entity
    • Link to collection group
    • Some idea of collection size / complexity?

Logic

A collection is deemed to be open if any organisation invited to return data for it has a deadline that hasn't elapsed (this includes extensions)

Table columns (proposed)

  • Collection name (link)
  • Collection group (link)
  • Opened?
  • Deadline
  • Extended*
  • Owner (name and email)*
  • Type (grants vs stats)
  • Submission method(s)
  • Certification required (yay/nay)
  • Associated organisations (invitees)
  • Returns submitted*
  • Records submitted*

* Do we have this info?
