What is being audited

We will be carrying out accessibility audits for the journeys (tasks) listed below. These involve 3 distinct user types across 2 different types of organisation, including both internal and external users.

Delta overview

Delta is the main platform the Ministry of Housing, Communities and Local Government (MHCLG) uses to collect data and make payments. Part of the department's Funding Service, it helps the department make policy decisions, release national statistics and process payments to local authorities.

1. Create dataset and instance

  • Clive Badger, MHCLG [internal user]
  • clive.badger@communities.gov.uk
  • Delta staging server login

Instructions: We want you to set-up a new collection by creating a dataset and instance.

Steps: User logs in → Set up data collection → manage datasets → create dataset button → enter selected info for specific dataset → link the dataset to the collection group “accessibility-audit-testing” → associate organisations → click add organisation → search and select the organisation “Rivendell Borough Council” → add button → navigate back to dataset management screen using the breadcrumbs (you need to press the dataset name twice) → set up or manage instances button → click new instance → name instance id/name and associate the form “access-testing-form-fun-form” → select collection type “collection” → select an instance period within the set dataset period → set a submission start/end date → continue → choose “record level” collection → continue → save the selected options on the data upload and delete options → return to the homepage using the breadcrumbs → select submit data → select test collection requests → find the collection you just created → select more → select test form → select testing complete → sign off collection instance → the collection is now ready for submissions
