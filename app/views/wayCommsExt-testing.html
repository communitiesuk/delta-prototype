Delta improvements

Hi Susie,

We are using your feedback to improve Delta. These improvements will make it easier and faster to use.

These are the first changes. You will be able to see them within the next few weeks.

Homepage

Your new homepage will show you:

  • deadlines you need to meet
  • open and missed collections
  • records that need certifying

Collection page

This is a new page for each collection. This will make it easier to see all relevant returns and records and find out how to get support.

There will be links to relevant documents such as guidance.

Record page

This is a new page to view a summary of a record. You can see this page by selecting a record on the collections page.

It will show:

  • the name of the record
  • the organisation
  • the collection
  • the status
  • who last updated/submitted/certified and when

How to use Delta

See our help and guidance pages for more information on how to use Delta.
