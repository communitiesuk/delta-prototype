Submissions dashboard

Submission reports for the collections you have access to. Sort this table by clicking on any column heading or use 'Show Filters' for more advanced filtering options.

Help with submission status
  • Collection name: the title of the collection.
  • Status: open, extended or closed. A collection is open if the collection window has not closed and no organisations have an open extension.
  • Deadline: the last date any organisation can submit a return. This includes extension deadlines.
  • Organisations: the number of organisations who have been invited to submit a return.
  • Not started: the number of organisations who have not started uploading data.
  • In progress: the number of returns containing one or more records either in draft format or requiring certification. These records require action before they can be submitted.
  • Completed: the number organisations that have submitted and certified a return. [OR for uncertifiable returns] the number organisations that have submitted a return
