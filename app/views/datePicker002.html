Design notes

  • Simplified the design by using a consistent radio button pattern for both "From Date" and "To Date".
  • Simplified error handling by requiring users to complete all date input fields (Day/Month/Year or Month/Year); inputs cannot be left empty.
  • Error messages:
    • Day is required.
    • Month is required.
    • Year is required.
    • Day, month, and year are required.
    • Year must be a 4-digit number.
    • "To Date" must be after "From Date".
  • The "Clear" link is displayed only if there are active filters within that accordion section.
  • The "Apply filters" button should display "Update filters" if filter changes have been made.
  • The "Reset filters" button is displayed only if:
    • A) Filters have been modified by the user, or
    • B) The page initially loaded with filters applied (pre-filtered).
