Delta improvements

We are using your feedback to improve Delta.

These improvements will take place over the next few months, and will:

  • improve data quality
  • reduce support requests
  • make it easier to access support
  • improve the audit trail

Homepage

Data providers will have a new homepage. It will show them:

  • open and missed collections
  • records that need certifying
  • what has been submitted
  • deadlines you need to meet

Collection page

There will be a page for each collection. This will make it easier for data providers to:

The permissions needed to view or certify data will not change.

Record page

This is a new page to view a summary of a record. Data providers can see this page by selecting a record on the collections page.

It will show:

  • the name of the record
  • the organisation
  • the collection
  • the status
  • who last updated/submitted/certified and when
